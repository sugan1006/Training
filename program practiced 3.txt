Overriding

program:

class Animal {
   public void move() {
      System.out.println("Animals can move");
   }
}

class Dog extends Animal {
   public void move() {
      super.move();   
      System.out.println("Dogs can bark and run");
   }
}

public class TestDog {

   public static void main(String args[]) {
      Animal b = new Dog();   
      b.move();  
   }
}

output:
Animals can move
Dogs can bark and run
